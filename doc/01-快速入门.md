## 1. 环境准备

**系统要求：**
- Python 3.10+（推荐 3.12）
- pip 或 uv 包管理器
- Git（可选）

**验证 Python 版本：**
```bash
python --version
# Python 3.12.x
```

## 2. 安装

```bash
# 方式一：开发模式安装（推荐学习时使用）
cd /path/to/apitest
pip install -e .

# 方式二：带 Web 管理界面
pip install -e ".[web]"

# 方式三：全量安装（含开发工具）
pip install -e ".[all]"
```

**验证安装成功：**
```bash
smartapi --version
# SmartAPI-Test, version 0.1.0
```

## 3. 项目初始化

```bash
# 创建新项目目录
mkdir my-api-tests && cd my-api-tests

# 自动初始化项目结构
smartapi init
```

初始化后的目录结构：
```
my-api-tests/
├── testcases/          # 测试用例目录
│   └── example.yaml    # 示例用例
├── environments/       # 环境配置目录
│   └── dev.yaml        # 开发环境配置
├── mock/               # Mock 配置目录
├── reports/            # 测试报告目录
└── conftest.py         # pytest fixtures
```

## 4. 编写第一个用例

创建文件 `testcases/hello.yaml`：

```yaml
# 我的第一个 API 测试用例
name: "Hello SmartAPI"
description: "验证 httpbin GET 接口"
base_url: "https://httpbin.org"
tags:
  - smoke
  - tutorial

steps:
  - name: "发送 GET 请求"
    request:
      method: GET
      url: "/get"
      params:
        hello: "smartapi"
        version: "1.0"
      headers:
        User-Agent: "SmartAPI-Test/0.1"
    asserts:
      # 验证状态码
      - target: status_code
        operator: eq
        expected: 200

      # 验证响应体字段
      - target: body
        expression: "$.args.hello"
        operator: eq
        expected: "smartapi"

      # 验证响应头
      - target: header
        expression: "Content-Type"
        operator: contains
        expected: "application/json"

      # 验证响应时间 < 5秒
      - target: response_time
        operator: lt
        expected: 5000
```

**用例结构说明：**

| 字段 | 说明 |
|------|------|
| `name` | 用例名称（必填） |
| `base_url` | 接口基础地址 |
| `tags` | 标签，用于筛选 |
| `steps` | 测试步骤列表（核心） |
| `steps[].request` | HTTP 请求配置 |
| `steps[].asserts` | 断言列表 |

## 5. 执行测试

SmartAPI-Test 提供三种执行方式：

### 方式一：CLI 命令（推荐入门）

```bash
# 执行单个用例
smartapi run testcases/hello.yaml

# 执行目录下所有用例
smartapi run testcases/

# 按标签筛选执行
smartapi run testcases/ --tags smoke

# 指定环境
smartapi run testcases/ --env dev
```

### 方式二：pytest 方式（推荐 CI/CD）

```bash
# 执行单个用例文件
pytest testcases/hello.yaml -v

# 执行所有用例
pytest testcases/ -v

# 指定 base_url
pytest testcases/ --smartapi-base-url https://httpbin.org

# 并发执行（需 pytest-xdist）
pytest testcases/ -n 4
```

### 方式三：Web 管理界面

```bash
# 启动 Web 服务
smartapi web --port 8100

# 打开浏览器访问
# http://127.0.0.1:8100
# 在「用例管理」页面点击 ▶ 按钮执行
```
